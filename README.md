# Appendix-Code
Effects of partial epitope coverage on the dynamics of antigenically diverse pathogen populations

## Contents
- `model1_nonoverlapping_complete_crossprotection`
  - Non-overlapping compartmental PEC/FEC model

- `model1_nonoverlapping_complete_crossprotection`
  - Ordered immune-history compartmental model with incomplete cross-immunity

- `model3_overlapping_incomplete_crossprotection`
  - Overlapping compartmental formulation with incomplete cross-protection

## Software

Code was developed in MATLAB

## Purpose

These scripts were used to simulate strain dynamics, bifurcation diagrams, calculate and visualise single-strain dominance under heterogeneous epitope targeting.
